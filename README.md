# Real-time Classroom Occupancy Monitoring

YOLOv3 based system for detecting:
- person
- chair
- bag

Used for estimating classroom occupancy.

## Dataset

Training dataset is built from:

- Filtered COCO dataset
- Classroom dataset

Classes:

0 person  
1 chair  
2 bag  

## Training

Using Darknet YOLOv3.
